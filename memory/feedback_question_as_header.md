When an assignment has numbered or listed prompts, write the exact question verbatim as the bold header for each section, followed by `<br>` and the answer on the next line:

```html
<p><strong>Do you think Gauguin fetishizes the figure of Teha'amana...?</strong><br>
Answer text here.</p>
```

**Why:** User confirmed this is their consistent style — they always write the question exactly as given. A paraphrased or shortened label is not acceptable.

**How to apply:** Any time an assignment has discrete prompts or questions, copy each prompt verbatim as the section label. Do not rephrase, shorten, or summarize. Apply to both essays and discussion posts when prompts are present.
